About

Weather Now shows current conditions and a 7-day forecast for any city, powered by the free Open-Meteo API — no API key required.

How it works

  • Your search is geocoded to coordinates.
  • Current weather and the daily forecast are fetched for those coordinates.
  • Your last city is remembered in the browser for next time.

No accounts, no tracking. Built with vanilla JavaScript.
