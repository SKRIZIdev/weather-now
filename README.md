# ⛅ Weather Now

Current conditions and a 7-day forecast for any city, powered by the free **Open-Meteo** API (no key needed). Your last city is remembered for next time.

**Live demo:** https://skrizidev.github.io/weather-now/

## Pages
- **Weather** — city search, current card (temp, humidity, wind), 7-day forecast
- **About** — how it works and data source

## Tech
Vanilla HTML / CSS / JavaScript. Geocoding + forecast via Open-Meteo; last city in `localStorage`.
